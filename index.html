Proyecto 1

Generador de QR

El Generador de Códigos QR es una herramienta web desarrollada para permitir a los usuarios crear fácilmente códigos QR personalizados para textos, URLs, información de contacto, eventos y más. Este proyecto fue creado con enfoque en la usabilidad y simplicidad, permitiendo al usuario ingresar datos y obtener en tiempo real un código QR descargable. El sistema puede ser utilizado para eventos, invitaciones, catálogos, tarjetas de presentación digitales, entre otros usos. También se adapta para funcionar como módulo integrable en sistemas más grandes como sistemas de asistencia, registro o verificación de identidad.

Proyecto 2

Calculadora de Losas

La Calculadora de Losa es una aplicación web interactiva diseñada para ayudar a ingenieros, albañiles y estudiantes de construcción a estimar de forma precisa los materiales necesarios para la elaboración de losas de concreto. El sistema permite al usuario seleccionar entre distintos tipos de estructura (losa regular, losa irregular, calle/pavimento o columna), ingresar las dimensiones y características del proyecto, y visualizar los resultados tanto en forma de lista como en gráficos y vista 3D. Para losas irregulares, la aplicación permite al usuario dibujar la forma directamente sobre un lienzo, calcular automáticamente el área, y luego estimar los materiales con base en el grosor proporcionado. El sistema calcula cantidades estimadas de cemento, arena, grava, agua y varillas según fórmulas preestablecidas, con el objetivo de brindar una guía confiable y práctica para planificación de obras.
